# Projeto Teste A/B:
Um dos três projetos finais do bootcamp de dados da Tripleten, este projeto realiza um teste A/B e um teste de hipótese para verificar se mudanças feitas em um site de vendas pela empresa responsáveis resultaram em aumento de 10% na taxa de conversões de usuários europeus do grupo B (product_page → product_card → purchase).

# Resultados:
O grupo de usuários B apresentou uma taxa de conversão negativa em comparação ao grupo A, indicando que as mudanças feitas no site não só resultaram em uma taxa de conversão menor do que um aumento em 10%, mas sim em uma perda de clientes percorrento todo o funil de vendas.

Seria ideal para a empresa no momento manter o site como ele estava para os usuários no grupo A.

# Tecnologias utilizadas:
Python

Pandas: Manipulação e tratamento de dados.

NumPy: Operações matemáticas.

Matplotlib, Seaborn, Plotly: Visualização de dados.

Proportions_ztest: Uso do teste Z para teste de hipótese.

Jupyter Notebook: Ambiente de desenvolvimento.
